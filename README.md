# mqtt
It's a kit of mqtt which based on mqtt-paho.

### mypublish.py
发布消息

### mysubscribe.py
订阅消息

### mqtt_kits.py
基于mqtt-paho封装的 mqtt工具箱

### cfgparser.py
配置文件解析，支持json\yaml\toml\ini\conf格式的配置文件

### mqtt_config.ini
mqtt broker (server) 配置信息

### 运行方式
先运行mysubscribe.py，再运行mypublish.py，可以看到完整mqtt信息流
